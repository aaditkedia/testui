# Documentation

Reference material for the templates in this repo. Nothing here is imported at runtime — the gallery reads from `src/config/` only.

## Build plans (`build-plans/`)

Engineering plans paired one-to-one with the spec sheets. Open these when you need to understand *how* a template was assembled or you're rebuilding from scratch.

- `pravah-rebuild-plan.md` — the AI-Lab / data-viz template.
- `BUILD-PLAN.md` — earlier umbrella plan (kept for history).
- `BERCO-BUILD-PLAN.md` — holding-company template.
- `NGPES-BUILD-PLAN.md` — fintech / payments template.
- `WESTSIDE-GRILL-BUILD-PLAN.md` — steakhouse restaurant template.
- `THE-SHELBY-BUILD-PLAN.md` — editorial restaurant template.
- `LAZEEZ-GRILL-BUILD-PLAN.md` — Mediterranean fast-casual template.
- `STUFFED-TO-THE-GRILLS-BUILD-PLAN.md` — cafe / catering template.

## Specs (`specs/`)

Original brand spec sheets. These are the "client-side" version of each template — what the original client's site needed to do. Generic gallery names in the sidebar are derived from these.

- `website-recreation-spec.md` — Pravah (AI lab).
- `ambassador-concept-b-cinematic.md` — Cinematic program concept.
- `ambassador-concept-c-editorial.md` — Editorial manifesto concept.
- `berco-spec.md`, `ngpes-spec.md`
- `westside-grill-spec.md`, `the-shelby-spec.md`, `lazeez-grill-spec.md`, `stuffed-to-the-grills-spec.md`

See `../CONTEXT.md` at the repo root for the overall gallery flow.
